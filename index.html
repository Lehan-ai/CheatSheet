My Cheat Sheet: CSS Visual Rules

Colors & Appearance

Tag Name Description
< background-color > Background Color Sets the background color
< color > Color Sets the text color
< opacity > Opacity Sets the transparancy level
< box-shadows > Box Shadow Creates a shadow around elements
< filter > Filter Creates visual effects like blur, brightness, grayscale

Backgrounds & Images

Tag Name Description
< background-image > Background Image Sets and image as the background
< background-size > Background Size Controls the scale of the background
< background-position > Background Position Is used to control where the image is palced

Borders & Outlines

Tag Name Description
< border > Border Shorthand for border width, style and color
< border-radius > Border Radius Rounds the corners of a border
< outline > Outline Creates a line around the element (does not affect layout)

Typography (Only Font Styles)

Tag Name Description
< font-size > Font Size Sets the size of the font
< font-family > Font Family Sets the font used such as Monospace. Fallback fonts can also be used in the same line after a comma.
< font-style > Font Style Sets Style of the font to italics for example
< font-weight > Font Weight Can either set text to bold or a scale of bold
